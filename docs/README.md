## About the Platform

**OXP** is a full-stack B2C e-commerce and community platform built for a sustainable materials brand. The platform supports multilingual content in **English**, **Korean**, and **Simplified Chinese**, and includes:

- A public marketing website
- A B2C product store with cart and checkout
- A community hub for user-generated content
- A B2B inquiry and lead management system
- A full-featured admin panel powered by Filament

**Technology Stack:**
- Backend: Laravel 13 (PHP 8.2+)
- Frontend: Next.js 16 (React 19, TypeScript)
- Admin Panel: Filament 5
- Database: MySQL 8+
- Storage: Azure Blob Storage (or local)
