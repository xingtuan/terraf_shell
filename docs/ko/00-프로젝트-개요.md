# 00 — 프로젝트 개요

## OXP란 무엇인가?

**OXP**는 지속 가능한 소재 브랜드를 위해 개발된 풀스택 다국어 B2C 및 B2B 디지털 플랫폼입니다. 이커머스 스토어, 커뮤니티 허브, 소재 라이브러리, B2B 문의 파이프라인을 하나의 통합 시스템으로 제공합니다.

내부 코드베이스명은 **Terraf**이며, 사용자에게 보이는 브랜드명은 **OXP**입니다.

---

## 비즈니스 목적

OXP 플랫폼은 다음을 위해 설계되었습니다:

1. **교육 및 브랜드 전달** — 브랜드의 소재 과학, 인증, 지속가능성을 글로벌 오디언스에게 소개합니다.
2. **직접 판매** — 현대적인 이커머스 스토어를 통해 고객이 제품을 탐색하고 구매할 수 있도록 합니다.
3. **커뮤니티 구축** — 등록 사용자가 아이디어, 프로젝트, 지식을 공유할 수 있는 커뮤니티를 운영합니다.
4. **B2B 리드 창출** — 도매, 유통, 제조, 샘플 요청, 파트너십 문의를 수집합니다.
5. **통합 관리** — 모든 콘텐츠, 사용자, 주문, 설정을 관리하는 강력한 관리자 패널을 제공합니다.

## 대상 사용자

| 사용자 유형 | 설명 |
|---|---|
| **일반 방문자** | 계정 없이 웹사이트, 소재, 기사, 제품, 커뮤니티를 탐색하는 사용자 |
| **등록 사용자** | 구매, 주문 관리, 커뮤니티 게시물 작성 및 상호작용이 가능한 인증 고객 |
| **B2B 파트너** | 도매, 파트너십, 협업 문의를 제출하는 기업 및 전문가 |
| **커뮤니티 크리에이터** | 커뮤니티 콘텐츠를 활발히 기여하는 등록 사용자 |
| **모더레이터** | 신고를 검토하고 콘텐츠를 관리하는 권한을 가진 신뢰할 수 있는 사용자 |
| **관리자** | 관리자 패널, 모든 콘텐츠, 사용자 관리, 설정에 완전한 접근 권한을 가진 플랫폼 운영자 |

## 지원 언어

| 언어 | 코드 | 범위 |
|---|---|---|
| 영어 | `en` | 기본 언어; 프론트엔드 및 관리자 패널 완전 지원 |
| 简体中文 (중국어 간체) | `zh` | 프론트엔드 완전 지원; 관리자 패널 부분 지원 |
| 한국어 | `ko` | 프론트엔드 완전 지원; 관리자 패널 부분 지원 |

번역 파일은 `B2C_frontend/messages/{locale}.json`에 저장됩니다. 관리자 패널은 `/admin/locale/{locale}`을 통해 언어를 전환할 수 있습니다.

## 주요 기능 모듈

### 1. 공개 웹사이트
마케팅 및 콘텐츠 레이어:
- **홈페이지** — 히어로 배너, 가치 제안, 소재 쇼케이스, 커뮤니티 미리보기, 협업 CTA
- **소재 라이브러리** — 사양, 스토리 섹션, 활용 사례, 인증이 포함된 상세 소재 페이지
- **기사** — 지식베이스 및 브랜드 블로그
- **법적 페이지** — 개인정보처리방침 및 이용약관 (관리자 패널에서 편집 가능)

### 2. 계정 시스템
- 회원가입, 로그인, 이메일 인증, 비밀번호 재설정
- 프로필 관리 (소개, 아바타, 위치)
- 배송 주소록
- 계정 대시보드

### 3. 스토어
- 제품 카탈로그 (카테고리, 검색, 필터)
- 제품 상세 페이지 (옵션, 이미지, 사양, FAQ)
- 장바구니 (비회원 및 회원)
- 결제 (배송비 계산 포함)
- 주문 추적

### 4. 커뮤니티
- 리치 텍스트 게시물 작성 (미디어 첨부 포함)
- 댓글, 답글, 좋아요, 저장
- 팔로우/팔로잉 관계
- 게시물 크라우드펀딩 캠페인 링크
- 콘텐츠 신고

### 5. B2B 및 문의
- 일반 문의 양식
- 구조화된 B2B 문의 양식
- 파트너십, 대학 협업, 제품 개발 문의 양식
- 샘플 요청 양식
- 관리자 패널 리드 파이프라인 관리

### 6. 관리자 패널
- 사용자 관리, 역할 할당, 계정 조치
- 모든 콘텐츠 완전 CRUD 관리
- 주문 처리 및 상태 관리
- 커뮤니티 콘텐츠 모더레이션
- 이메일 센터 (템플릿, 이벤트, 로그)
- 설정 시스템 (앱, 메일, 스토리지, 배송, 세금, 커뮤니티 정책)

## 알려진 제한 사항

- **결제 게이트웨이**: 결제 필드는 존재하지만 실제 게이트웨이가 연결되어 있지 않습니다. 주문은 "미결제" 상태로 생성되며 수동 확인이 필요합니다.
- **NZ Post 배송**: 실시간 견적이 구현되어 있지만 기본적으로 비활성화 상태입니다. 기본 고정 요금 규칙이 적용됩니다.
- **크라우드펀딩 캠페인**: 백엔드 데이터 모델과 관리자 관리는 완료되었지만, 프론트엔드 캠페인 현황 표시는 제한적입니다.
- **이메일**: 이메일 템플릿 및 이벤트 시스템은 구축되어 있지만, 프로덕션 환경에서 이메일을 발송하려면 SMTP 서버 설정이 필요합니다.
- **관리자 패널 UI 현지화**: 관리자 패널 인터페이스는 주로 영어로 제공됩니다.
