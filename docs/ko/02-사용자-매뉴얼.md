# 02 — 사용자 매뉴얼

## 소개

이 매뉴얼은 OXP 플랫폼의 일반 사용자를 위해 작성되었습니다. 공개 웹사이트, 스토어, 커뮤니티 허브를 이용하는 고객 및 방문자를 대상으로 합니다.

기술적인 지식은 필요하지 않습니다. 플랫폼은 직관적이고 접근하기 쉽도록 설계되었습니다.

---

## 1. 웹사이트 방문

웹 브라우저에서 플랫폼 운영자가 제공한 OXP 웹사이트 URL을 입력하면 홈페이지가 자동으로 열립니다.

홈페이지 구성:
- 브랜드 헤드라인과 CTA 버튼이 있는 히어로 섹션
- 소재의 중요성을 설명하는 "Why It Matters" 섹션
- 핵심 가치와 비즈니스 기둥
- 소재 쇼케이스 섹션
- 커뮤니티 아이디어 미리보기
- 협업 및 파트너십 초대
- 최종 CTA 섹션

---

## 2. 언어 전환

플랫폼은 **영어 (English)**, **한국어**, **중국어 간체 (简体中文)**를 지원합니다.

전환 방법:
1. 상단 내비게이션 바에서 **언어 전환기**를 찾습니다 (지구본 아이콘 또는 언어 약어로 표시됨).
2. 드롭다운에서 원하는 언어를 선택합니다.
3. 선택한 언어로 페이지가 다시 로드됩니다.

언어 설정은 URL에 반영됩니다 (예: `/en/`, `/ko/`, `/zh/`).

---

## 3. 웹사이트 내비게이션

상단 내비게이션 바:

| 섹션 | 설명 |
|---|---|
| **홈** | 홈페이지로 이동 |
| **소재** | OXP 소재 라이브러리 보기 |
| **스토어** | 제품 탐색 및 구매 |
| **커뮤니티** | 커뮤니티 게시물 보기 및 작성 |
| **기사** | 지식베이스 기사 읽기 |
| **문의/B2B** | 문의 또는 비즈니스 연락 제출 |
| **계정** | 로그인, 회원가입 또는 계정 관리 |

---

## 4. 소재 라이브러리

소재 페이지에서 확인 가능한 내용:
- 소재 이름, 설명, 지속가능성 스토리
- 기술 사양
- 실제 활용 사례
- 인증 및 품질 증명
- 비주얼 갤러리 및 스토리 섹션

---

## 5. 스토어

### 5.1 제품 탐색

1. 내비게이션 바에서 **스토어**를 클릭합니다.
2. 제품 그리드를 탐색합니다.
3. **카테고리 필터**로 제품 유형별로 필터링합니다.
4. **검색창**으로 이름 또는 키워드로 검색합니다.

### 5.2 제품 상세 페이지

제품 카드를 클릭하면 상세 페이지가 열립니다:
- 제품 이름 및 설명 (선택한 언어로 표시)
- 제품 이미지 갤러리
- 제품 옵션 (크기, 색상 등 선택 가능 시)
- 가격 (NZD, GST 포함)
- 재고 상태
- 특징, 관리 방법, 소재 장점
- 인증 정보
- 배송 및 반품 정보
- 제품 FAQ
- **장바구니 담기** 버튼 (문의 전용 제품은 문의 버튼)
- **샘플 요청** 버튼 (해당 제품에 활성화된 경우)

### 5.3 장바구니에 담기

1. 제품 상세 페이지에서 옵션을 선택합니다 (필요 시).
2. **장바구니 담기**를 클릭합니다.
3. 사이드바에 담기 확인 메시지가 나타납니다.
4. 계속 쇼핑하거나 결제를 진행합니다.

### 5.4 결제

> **참고**: 비회원 결제와 회원 결제 모두 지원됩니다.

**비회원 결제:**
1. 장바구니에서 **결제 진행**을 클릭합니다.
2. 이메일 주소와 배송 정보를 입력합니다.
3. 배송 방법을 선택합니다.
4. 주문을 확인하고 제출합니다.

**회원 결제:**
1. 로그인 상태인지 확인합니다.
2. 장바구니에서 **결제 진행**을 클릭합니다.
3. 저장된 주소를 선택하거나 새 주소를 입력합니다.
4. 배송 방법을 선택하고 주문을 확인합니다.

### 5.5 주문 상태

| 상태 | 의미 |
|---|---|
| **처리 대기 중** | 주문 접수, 확인 대기 |
| **확인 완료** | 스토어에서 주문 확인 |
| **처리 중** | 주문 준비 중 |
| **배송 완료** | 발송 완료 |
| **배달 완료** | 수령 완료 |
| **취소됨** | 주문 취소 |

---

## 6. 회원가입

1. 내비게이션에서 **회원가입**을 클릭합니다.
2. **이름**, **사용자명**, **이메일**, **비밀번호**를 입력합니다.
3. **계정 만들기**를 클릭합니다.
4. 이메일로 전송된 **인증 링크**를 클릭하여 이메일을 인증합니다.
5. 인증 완료 후 로그인 가능합니다.

---

## 7. 로그인

1. 내비게이션에서 **로그인**을 클릭합니다.
2. **이메일**과 **비밀번호**를 입력합니다.
3. **로그인**을 클릭합니다.

비밀번호를 잊은 경우, 로그인 페이지에서 **비밀번호 찾기**를 클릭하여 재설정 이메일을 요청하세요.

---

## 8. 계정 대시보드

### 8.1 프로필 관리
**계정 → 프로필**에서:
- **표시 이름**, **소개**, **위치** 업데이트
- **아바타** 업로드 또는 변경

### 8.2 주소 관리
**계정 → 주소**에서:
- 저장된 배송 주소 보기
- 새 주소 추가
- 기본 주소 설정 (결제 시 자동 입력)
- 주소 삭제

### 8.3 주문 내역
**계정 → 주문**에서:
- 모든 과거 주문 보기
- 주문 클릭 시 상세 내용 확인 (상품, 수량, 가격, 현재 상태)

---

## 9. 커뮤니티

### 9.1 게시물 탐색

1. 내비게이션에서 **커뮤니티**를 클릭합니다.
2. 게시물 피드를 탐색합니다.
3. **검색창**으로 키워드 검색합니다.
4. **카테고리 필터**로 특정 카테고리를 봅니다.
5. 게시물 제목이나 카드를 클릭하여 전체 내용을 읽습니다.

### 9.2 게시물 작성 (로그인 필요)

1. **게시물 작성**을 클릭합니다.
2. 다음 내용을 입력합니다:
   - **제목** (필수)
   - **내용** — 리치 텍스트 편집기로 형식화된 내용 작성
   - **커버 이미지** 업로드
   - **카테고리** 선택
   - **태그** 추가
   - **미디어 첨부** — 최대 12개 파일 또는 4개 외부 링크
3. **게시**를 클릭하여 제출합니다.

### 9.3 좋아요 및 저장

- 게시물의 **좋아요 아이콘**을 클릭하여 좋아요를 누릅니다.
- **북마크 아이콘**을 클릭하여 저장 목록에 추가합니다.
- **계정 → 커뮤니티 → 저장됨**에서 저장한 게시물을 확인합니다.

### 9.4 댓글

1. 게시물 하단 댓글 섹션으로 스크롤합니다.
2. 텍스트 박스에 댓글을 입력합니다.
3. **댓글 게시**를 클릭하여 제출합니다.
4. 특정 댓글에 답글을 달려면 해당 댓글 아래 **답글**을 클릭합니다.

### 9.5 사용자 팔로우

다른 사용자의 사용자명을 클릭하여 프로필 페이지로 이동한 후 **팔로우**를 클릭합니다.

### 9.6 알림

내비게이션의 **알림 벨 아이콘**을 클릭하여 알림을 확인합니다. 다음 경우에 알림을 받습니다:
- 누군가 내 게시물에 좋아요를 눌렀을 때
- 누군가 내 게시물에 댓글을 달았을 때
- 누군가 나를 팔로우했을 때
- 시스템 공지가 게시되었을 때

### 9.7 콘텐츠 신고

부적절한 콘텐츠를 발견한 경우:
1. 게시물 또는 댓글의 **더보기 메뉴 (⋮)**를 클릭합니다.
2. **신고**를 선택합니다.
3. 신고 이유를 선택합니다.
4. 제출합니다.

---

## 10. B2B 문의

비즈니스 관련 문의 (도매, 파트너십, 샘플 요청):
1. 내비게이션이나 푸터에서 **B2B** 또는 **문의하기**로 이동합니다.
2. 구조화된 문의 양식을 작성합니다.
3. 제출합니다.
4. 확인 메시지가 표시되면 플랫폼 팀이 검토 후 연락드립니다.

---

## 11. 일반적인 오류 메시지

| 메시지 | 의미 |
|---|---|
| "계속하려면 로그인하세요" | 인증이 필요한 작업입니다 |
| "계정이 제한되었습니다" | 게시물/댓글 작성 불가 상태입니다 |
| "계정이 정지되었습니다" | 계정이 일시 정지되었습니다 |
| "이 게시물을 이용할 수 없습니다" | 게시물이 삭제되었거나 검토 중입니다 |
| "품절" | 선택한 제품 옵션의 재고가 없습니다 |
| "링크가 유효하지 않거나 만료되었습니다" | 비밀번호 재설정 또는 이메일 인증 링크가 만료되었습니다 |
