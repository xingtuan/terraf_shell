# 00 — 项目总览

## OXP 是什么？

**OXP** 是一个面向可持续材料品牌的全栈多语言 B2C 及 B2B 数字平台。平台整合了电商商城、社区内容中心、素材知识库和 B2B 询盘管道，为客户、创作者和商业合作伙伴提供统一的数字服务体验。

代码库内部名称为 **Terraf**，用户看到的品牌名称为 **OXP**。

---

## 商业目的

OXP 平台的设计目标：

1. **教育与品牌传播** — 向全球受众展示品牌的材料科学、认证体系和可持续发展理念。
2. **直销** — 让消费者通过现代化商城直接浏览、购买产品，支持购物车、结账和订单追踪。
3. **建设社区** — 允许注册用户分享想法、项目和知识，支持社区帖子、评论、点赞和关注。
4. **B2B 线索获取** — 收集批发、分销、生产、材料申请和合作伙伴询盘。
5. **集中管理** — 为管理员提供强大的后台，统一管理平台内容、用户、订单和配置。

## 目标用户

| 用户类型 | 描述 |
|---|---|
| **公开访客** | 无需账号即可浏览网站、材料库、文章、产品和社区内容 |
| **注册用户** | 已登录客户，可下单、管理订单、发布社区帖子及互动 |
| **B2B 合作伙伴** | 提交批发、合作或定制生产询盘的企业和专业人士 |
| **社区创作者** | 积极贡献社区内容（帖子、评论、项目展示）的注册用户 |
| **版主** | 拥有内容审核权限的可信用户，可处理举报和内容管理 |
| **管理员** | 拥有完整后台访问权限的平台运营者，可管理所有内容、用户和系统设置 |

## 支持语言

平台全面支持三种语言，覆盖所有用户界面：

| 语言 | 代码 | 覆盖范围 |
|---|---|---|
| 英语 | `en` | 默认语言；前台和管理后台均完整支持 |
| 简体中文 | `zh` | 前台完整支持；管理后台部分本地化 |
| 한국어（韩语） | `ko` | 前台完整支持；管理后台部分本地化 |

翻译文件存储于 `B2C_frontend/messages/{locale}.json`，后端 API 在支持多语言字段时按请求语言返回内容。管理后台支持通过 `/admin/locale/{locale}` 切换界面语言。

## 主要功能模块

### 1. 公开网站
平台的营销与内容展示层：
- **首页** — 主视觉横幅、价值主张、材料展示、社区预览、合作 CTA
- **材料库** — 详细的材料页面，包含规格、故事章节、应用场景和认证
- **文章** — 知识库与品牌博客，富文本内容
- **法律页面** — 隐私政策和使用条款，可由后台直接编辑

### 2. 账号系统
用户身份验证与个人资料管理：
- 注册、登录、邮箱验证、密码重置
- 个人资料管理（简介、头像、地址）
- 收货地址管理
- 账号中心（社区动态 + 订单历史）

### 3. 商城
完整的电商功能：
- 商品目录（分类、搜索、筛选）
- 商品详情页（变体、图片、规格、FAQ）
- 购物车（游客和登录用户均支持）
- 结账（含运费计算）
- 订单追踪（游客和注册用户）

### 4. 社区
用户原创内容平台：
- 富文本帖子创作（含媒体附件）
- 评论、回复、点赞、收藏
- 关注/被关注关系
- 帖子上的众筹（资助）链接
- 内容搜索
- 举报不当内容

### 5. B2B 与联系
商业线索获取与询盘管理：
- 通用联系表单
- 结构化 B2B 询盘表单
- 合作伙伴、高校合作和产品开发询盘表单
- 材料申请表单
- 管理后台审核与线索跟进

### 6. 管理后台
基于 Filament 5 (Laravel) 构建：
- 用户管理、角色设置和账号审核
- 所有内容的完整 CRUD（产品、材料、文章、首页板块、法律页面）
- 订单处理和状态管理
- 社区内容审核（举报、帖子/评论状态、用户限制）
- 邮件中心（模板、事件、日志）
- 设置系统（应用、邮件、存储、运费、税率、社区策略）
- 交付就绪检查清单
- 初始内容导入/清除工具

## 已知限制

- **支付网关**：订单模型记录了支付方式和参考号，但未接入真实支付网关（如 Stripe、PayPal 等）。订单以"未支付"状态创建，需人工确认付款。
- **新西兰邮政运费**：实时运费查询已实现但默认禁用，默认使用固定费率规则。
- **众筹活动**：后端数据模型和后台管理完整，但前台活动状态/进度展示有限。
- **邮件发送**：邮件模板和事件系统已完整构建，但在生产环境需配置 SMTP 服务器才能正常发送邮件。
- **管理后台界面本地化**：管理后台界面主要为英文。后台语言切换仅影响部分标签。
