# 02 — 用户使用说明

## 简介

本手册面向 OXP 平台的终端用户——包括访客、消费者和社区成员。无需任何技术背景，平台设计直观易用。

---

## 1. 访问网站

在浏览器中输入平台运营方提供的 OXP 网站地址，即可打开首页。

首页包含：
- 主视觉横幅与号召性用语按钮
- "为何重要"版块，阐释材料的意义
- 商业支柱与核心价值主张
- 材料展示版块
- 社区创意预览
- 合作与伙伴关系邀请
- 最终号召性行动

---

## 2. 切换语言

平台支持**英语（English）**、**韩语（한국어）**和**简体中文**。

切换方法：
1. 在顶部导航栏找到**语言切换器**（通常显示为地球图标或语言缩写）。
2. 在下拉菜单中选择目标语言。
3. 页面将以所选语言重新加载。

语言偏好会体现在 URL 中（如 `/en/`、`/ko/`、`/zh/`）。

---

## 3. 网站导航

顶部导航栏提供以下快速入口：

| 栏目 | 说明 |
|---|---|
| **首页** | 返回首页 |
| **材料** | 查看 OXP 材料库 |
| **商城** | 浏览和购买产品 |
| **社区** | 查看和发布社区内容 |
| **文章** | 阅读知识库文章 |
| **联系 / B2B** | 提交询盘或商务联系 |
| **账号** | 登录、注册或管理账号 |

---

## 4. 材料库

材料页面展示品牌旗舰可持续材料。您可以查看：
- 材料名称、描述和可持续发展故事
- 技术规格参数
- 现实应用案例
- 认证和质量证明
- 视觉图库和叙事章节

导航路径：点击顶部菜单的**材料**。

---

## 5. 文章

文章是平台团队创作的教育性和信息性内容。浏览方式：
1. 点击导航栏中的**文章**，或在首页找到文章板块。
2. 浏览文章列表（按发布日期排序）。
3. 点击任意文章标题阅读全文。

文章包含封面图片、预计阅读时间和富文本内容。

---

## 6. 商城

### 6.1 浏览商品

1. 点击导航栏中的**商城**。
2. 浏览商品网格。
3. 使用左侧（或移动端顶部）的**分类筛选器**按商品类别过滤。
4. 使用**搜索栏**按名称或关键词搜索商品。
5. 使用**排序选项**按价格、最新或相关性排序。

### 6.2 商品详情页

点击任意商品卡片打开详情页，您可以看到：

- **商品名称和描述**（显示您所选的语言）
- **商品图片** — 点击缩略图切换主图
- **商品变体** — 如有，请选择尺寸、颜色等选项
- **价格** — 以新西兰元（NZD）显示，含 GST
- **库存状态** — 显示商品是否有货
- **特性和护理说明**
- **材料优点**
- **认证信息**
- **配送和退货说明**
- **商品 FAQ**
- **加入购物车**按钮（询价商品则显示询盘按钮）
- **材料申请**按钮（如该商品已启用）

### 6.3 加入购物车

1. 在商品详情页，如需选择变体（尺寸、颜色等），请先完成选择。
2. 点击**加入购物车**。
3. 侧边栏弹出确认提示。
4. 继续购物或前往结账。

### 6.4 查看和管理购物车

点击顶部导航的**购物车图标**打开侧边栏，或访问 `/store/cart`。

购物车中可：
- 查看所有商品及数量和价格
- 使用 `+` 和 `−` 按钮调整数量
- 点击删除按钮移除商品
- 查看小计金额
- 继续结账

### 6.5 结账

> **说明**：支持游客结账和注册用户结账。

**游客结账：**
1. 在购物车页面点击**前往结账**。
2. 输入邮箱地址和收货信息。
3. 选择运费方式（标准或快递，如适用还有乡村附加费）。
4. 确认订单摘要。
5. 输入支付信息（由运营方配置支付网关）。
6. 提交订单。

**注册用户结账：**
1. 确保已登录。
2. 将商品加入购物车后点击**前往结账**。
3. 选择已保存地址或输入新地址。
4. 选择运费方式。
5. 确认并提交订单。

### 6.6 订单确认

成功下单后：
- 显示**订单确认页面**，包含订单编号（格式：`OXP-XXXXXX`）。
- 如已配置邮件，将收到确认邮件。
- 使用订单编号追踪订单状态。

### 6.7 追踪订单

**游客：**
1. 前往**商城 → 订单**或直接访问 `/store/orders`。
2. 输入邮箱地址和订单编号。
3. 查看订单状态和商品明细。

**注册用户：**
1. 进入**账号 → 订单**。
2. 在列表中找到您的订单。
3. 点击订单查看完整详情和状态。

### 订单状态说明

| 状态 | 含义 |
|---|---|
| **待处理** | 已收到订单，等待确认 |
| **已确认** | 商家已确认订单 |
| **处理中** | 订单正在准备中 |
| **已发货** | 订单已发出 |
| **已送达** | 订单已签收 |
| **已取消** | 订单已取消 |

---

## 7. 账号注册

1. 点击导航栏中的**注册**。
2. 填写**姓名**、**用户名**、**邮箱地址**和**密码**。
3. 点击**创建账号**。
4. 查收邮件中的**验证链接**，点击完成邮箱验证。
5. 验证完成后即可登录。

---

## 8. 登录

1. 点击导航栏中的**登录**。
2. 输入**邮箱地址**和**密码**。
3. 点击**登录**。

忘记密码时，在登录页面点击**忘记密码？**，申请密码重置邮件。

---

## 9. 账号中心

登录后，通过顶部导航进入账号中心。仪表盘显示：
- 欢迎信息和用户名
- 订单、社区动态和地址的快捷入口
- 账号状态提醒（如账号存在限制）

### 9.1 个人资料管理

进入**账号 → 个人资料**：
- 更新**显示名称**、**个人简介**和**所在地**
- 上传或更换**头像**

### 9.2 账号设置

进入**账号 → 设置**：
- 更改**邮箱地址**
- 更改**密码**

### 9.3 地址管理

进入**账号 → 地址**：
- 查看已保存的收货地址
- 添加新地址
- 编辑现有地址
- 设置默认地址（结账时自动填充）
- 删除地址

### 9.4 订单历史

进入**账号 → 订单**：
- 查看所有历史订单
- 点击订单查看完整详情（商品、数量、价格和当前状态）

---

## 10. 社区

### 10.1 浏览社区帖子

1. 点击导航栏中的**社区**。
2. 浏览帖子动态流。
3. 使用**搜索栏**通过关键词查找帖子。
4. 使用**分类筛选**查看特定类别的帖子。
5. 点击帖子标题或卡片阅读全文。

### 10.2 创建帖子（需登录）

> 必须登录才能发帖。

1. 点击**创建帖子**。
2. 填写以下内容：
   - **标题**（必填）
   - **内容** — 使用富文本编辑器撰写格式化内容
   - **封面图片** — 上传帖子顶部显示的图片
   - **分类** — 选择最相关的分类
   - **标签** — 添加标签提高可发现性
   - **媒体附件** — 最多附加 12 个文件或 4 个外部链接
3. 点击**发布**提交帖子。

### 10.3 点赞帖子

点击帖子上的**点赞图标**进行点赞，再次点击取消点赞。点赞数公开显示。

### 10.4 收藏帖子

点击**书签图标**将帖子保存到个人收藏列表。在**账号 → 社区 → 已收藏**查看收藏内容。

### 10.5 评论

1. 滚动至帖子底部的评论区。
2. 在文本框中输入评论内容。
3. 点击**发表评论**提交。
4. 回复特定评论时，点击该评论下的**回复**。

您可以编辑或删除自己发布的评论。

### 10.6 关注用户

点击平台上任意用户名查看其公开主页，在主页上点击**关注**即可关注该用户。

### 10.7 通知

点击导航栏中的**通知铃铛图标**查看通知。以下情况会收到通知：
- 有人点赞您的帖子
- 有人评论您的帖子
- 有人回复您的评论
- 有人关注了您
- 平台发布系统公告

### 10.8 举报内容

若发现违规内容：
1. 点击帖子或评论上的**三点菜单（⋮）**。
2. 选择**举报**。
3. 选择举报原因。
4. 提交举报。

举报将由版主和管理员审核处理。

---

## 11. B2B 询盘

若您是希望批发、合作或申请评估材料的企业：
1. 前往导航或页脚中的 **B2B** 或**联系我们**。
2. 填写结构化询盘表单（公司信息、联系方式、需求说明）。
3. 提交表单。
4. 显示确认消息。平台团队将审核您的询盘并联系您。

---

## 12. 常见错误提示说明

| 提示信息 | 含义 |
|---|---|
| "请先登录以继续" | 需要登录才能执行此操作 |
| "您的账号已被限制" | 账号无法发帖或评论，请联系客服 |
| "您的账号已被暂停" | 账号已临时停用 |
| "该帖子不可用" | 帖子已被删除或正在审核 |
| "商品已售罄" | 所选商品变体无剩余库存 |
| "链接无效或已过期" | 密码重置或邮箱验证链接已过期，请重新申请 |
