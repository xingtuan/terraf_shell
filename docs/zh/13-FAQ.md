# 13 — 常见问答（FAQ）

---

## A. 普通用户常见问题

### A1. 浏览网站需要注册账号吗？

不需要。公开内容——首页、材料库、文章、社区帖子和商品目录——均可在无账号状态下浏览。购物下单、发布社区帖子、互动（点赞、评论）以及收藏帖子需要账号。

### A2. 如何切换网站语言？

在顶部导航栏找到语言切换器，平台支持**英语**、**韩语（한국어）**和**简体中文**。

### A3. 可以不注册账号直接下单吗？

可以。平台支持游客结账，需提供邮箱地址和收货信息。下单后可通过订单编号和邮箱地址在**商城 → 订单**页面查询订单。

### A4. 如何追踪我的订单？

- **已登录**：进入**账号 → 订单**查看所有订单。
- **游客结账**：前往**商城 → 订单**，输入订单编号（格式：`OXP-XXXXXX`）和结账邮箱。

### A5. 商城使用什么货币？

商城以**新西兰元（NZD）**运营，价格含 15% GST（新西兰商品及服务税）。

### A6. 如何重置密码？

在登录页面点击**忘记密码？**，输入邮箱地址提交。系统将发送密码重置链接，点击链接设置新密码。链接有有效期，过期后需重新申请。

### A7. 为什么我的帖子显示"待审核"？

根据平台当前的审核设置，新帖子可能需要经过审核才能公开显示。版主将审核您的帖子并决定是否批准发布。

### A8. 如何举报不当内容？

点击帖子或评论上的三点菜单（⋮），选择**举报**，选择原因并提交。版主将审核处理。

### A9. 每篇帖子最多可以上传多少文件？

最多可附加 **12 个文件**（图片和文档）和 **4 个外部链接**，单个文件不超过 10 MB。

### A10. 可以删除我的账号吗？

目前用户无法通过前台自助删除账号。如需删除账号，请联系平台管理员。

## B. 管理员常见问题

### B1. 如何访问管理后台？

访问 `{后端URL}/admin`，使用具有 `admin` 角色的账号登录。

### B2. 如何创建新的管理员用户？

1. 正常注册或创建用户账号。
2. 在管理后台前往**用户 → 用户**。
3. 找到该用户并编辑记录。
4. 将**角色**改为**管理员**并保存。

### B3. 如何在正式上线前删除演示内容？

前往**管理后台 → 系统 → 演示数据清除**，查看演示内容列表后点击**删除演示内容**。此操作不可撤销。

### B4. 如何更新隐私政策/使用条款？

前往**管理后台 → 系统 → 法律页面设置**，分别编辑三种语言的内容，保存后立即在网站生效。

### B5. 如何配置邮件发送？

前往**管理后台 → 系统 → 邮件设置**，填写 SMTP 凭据后点击**发送测试邮件**验证，设置立即生效。

### B6. 如何将社区帖子设为精选？

在管理后台**社区 → 帖子**中找到目标帖子并编辑，勾选**精选**复选框保存即可。

### B7. 如何导出 B2B 线索？

前往**管理后台 → B2B/线索 → B2B 线索**，点击**导出**按钮下载 CSV 文件。

### B8. 如何处理后台失败任务？

在终端执行：
```bash
php artisan queue:failed        # 查看失败任务
php artisan queue:retry all     # 重试全部失败任务
php artisan queue:flush         # 清除失败任务（调查后执行）
```
或在管理后台**系统 → 交付就绪**页面查看失败任务摘要。

## C. 技术维护人员常见问题

### C1. 如何运行数据库迁移？

```bash
cd B2C_backend
php artisan migrate
```

查看状态：`php artisan migrate:status`

### C2. 如何检查 i18n 翻译键覆盖率？

```bash
cd B2C_frontend
npm run check:i18n
```

### C3. 日志文件在哪里？

`B2C_backend/storage/logs/laravel.log`

### C4. 如何添加新翻译键？

1. 先在 `B2C_frontend/messages/en.json` 添加该键。
2. 同步添加至 `ko.json` 和 `zh.json`。
3. 运行 `npm run check:i18n` 验证覆盖。
4. 重新构建前端。

### C5. 如何验证存储链接是否正常？

```bash
ls -la B2C_backend/public/storage
# 应显示：storage -> /path/to/storage/app/public
```

缺失时运行：`php artisan storage:link`

## D. 客户/业务负责人常见问题

### D1. 能更改平台名称"OXP"吗？

可以。更新后端 `.env` 中的 `APP_NAME`，更新前端品牌资源，以及所有引用平台名称的邮件模板。完整品牌重塑请咨询开发团队。

### D2. 可以再增加语言吗？

可以，但需要开发工作：添加新翻译文件、更新路由配置、为新语言添加数据库列，以及更新管理后台的编辑界面。

### D3. 包含支付网关吗？

不包含。当前平台记录订单和支付意向，但不处理实际支付。需要集成第三方支付网关（如 Stripe、PayPal 或本地支付服务商），这需要额外开发工作。

### D4. 数据归属于谁？

所有平台数据存储在您的 MySQL 数据库和 Azure Blob Storage（或本地存储）中，您完全拥有所有数据的所有权。

### D5. 如何备份平台？

详见 [11-部署与维护说明.md](./11-部署与维护说明.md)。核心要点：
- 每日备份 MySQL 数据库
- 定期备份 Azure Blob Storage（或本地媒体文件）
- 安全备份 `.env` 文件
